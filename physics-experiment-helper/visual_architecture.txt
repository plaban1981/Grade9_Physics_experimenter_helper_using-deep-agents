DATA FLOW SEQUENCE
==================

1. USER REQUEST
   User Input → Web Frontend → FastAPI → Main Agent

2. PLANNING
   Main Agent → write_todos → Task Breakdown

3. RESEARCH
   Main Agent → Research Sub-Agent → Web Search → Resources

4. GENERATION
   Main Agent → File Tools → Markdown Files → Real-time Updates

5. REVIEW
   Main Agent → Critique Sub-Agent → Quality Check

6. DELIVERY
   Main Agent → File Organization → Export → User
